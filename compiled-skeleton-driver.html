In mt7610u.

I copied the mt7601 driver from here to a local repo. I need kernel-devel and kernel-headers to compile the driver. I stripped out most of the source files except for usb.c, usb.h, and mt7601.h. I kept the usb.* files because the entry point for the driver is there, module_usb_driver. I kept mt7601.h, now renamed to mt7610.h, because it seemed to define a lot of the structs and a few inline functions that are used in usb.*. I also kept the Makefile.

When I tried to compile the skeleton, it produced nothing. This is because I needed to modify the Kbuild file so that it produced an mt7610 instead of an mt7601 driver.
